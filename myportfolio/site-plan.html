Site Plan for My Portfolio

Project Requirements

  • The personal site must consist of a home page and at least one child page.
  • The website must be hosted on the web.
  • Each page must have significant textual content, at least one image, and working links to child pages.
  • The site must be laid out and styled according to wireframes, be visually appealing and readable, and valid HTML and CSS.
  • The site should score at least 97 on Lighthouse.
  • JavaScript should include: organization, DOM interaction, event handling, conditional branching, objects, ES Modules, arrays, and array methods.

Purpose and Audience

The purpose of this site is to showcase my portfolio, including projects, skills, and experiences. The intended audience is potential employers and clients who are interested in my work.

Description of Dynamic Elements

JavaScript will be used to create interactive elements such as a dynamic project gallery where users can filter projects by category. Functions will manage these interactions, including event listeners for filtering options and updating the displayed content.
